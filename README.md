# ESP32 AI Voice Robot (Xiaozhi Dev Integration)

This repository contains the ongoing development and firmware architecture for an AI-powered Voice Robot built using the **ESP32** microcontroller and integrated with the [Xiaozhi AI platform](https://xiaozhi.dev/).

## 🚀 Project Status
* ⏳ **Phase 1: Project Initialization & Repository Setup** (Current)
* 🛠️ **Phase 2:** Firmware Architecture & WebSocket Protocols
* 🔌 **Phase 3:** Hardware Integration (INMP441 Mic & MAX98357A DAC)
* 🧠 **Phase 4:** Custom LLM Orchestration via Xiaozhi Dev Console

## 🛠️ Planned Hardware Setup
* **Microcontroller:** ESP32-WROOM-32E or ESP32-S3
* **Audio Input:** INMP441 I2S Microphone
* **Audio Output:** MAX98357A I2S DAC Module + Speaker
